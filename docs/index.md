# calctl Documentation

Welcome to calctl - a command-line calendar management tool.

## Quick Links

- [GitHub Repository](https://github.com/chichizhang0510/calctl)
- [Docker Hub](https://hub.docker.com/r/chichizhang/calctl)

## Features

- Event management (add, edit, delete)
- Conflict detection
- Recurring events support
- Multiple view modes (list, agenda)
- JSON output support

## Installation
```bash
pip install calctl
```
